Lemma Tab

Lemma tab provides information about different properties of the lemma. These are the following:

Comment

Any comment to the Lemma which did not fit in any other tab.

Comment (english translation)

Any comment to the english translation of the Lemma.

lsort, arb_vermerk, simplify, wclassnum, woart, word_class are all information which were transformed from the old program and are no longer relevant.
