Object Tab

This tab is specific for an object, it does not appear when you are working on a text.

In this tab there are six input fields where you can add metadata for your object by choosing entries from the Thesaurus.

The sectuion "Description of object" contains:

  • Type of Object choose the type of your object (eg. Schriftrolle for a scroll: entry 21- Objekttyp)
  • Object is component
  • Owner
  • Description
  • Comment allows you to write your own comment on the object.

  • There are also four check boxes available : Model, Imitation, Miniature, Skeuomorph

  • Technical details: Material, Dimentions, Condition, Technique, Comment.
  • Archaelogical and cultural context of object: grouping, cultural context, comment

There are also five boxes available to check: Model, Imitation, Miniature, Skeuomorph
